This project showcases an interactive Sales & Profit Analysis Dashboard built using Power BI. It provides a comprehensive view of sales performance across different countries, customer segments, and products.

The dashboard helps in identifying key revenue drivers, high-performing markets, and profitable segments to support business decision-making.

🎯 Objectives
1.Analyze total sales and profit performance
2.Compare country-wise and segment-wise performance
3.Identify top-selling products
4.Track sales and profit trends over time
5.Enable interactive filtering for better insights

📈 Key Insights
1.Government and Enterprise segments contribute the highest sales
2.United States and Canada are the top-performing countries
3.Product Paseo generates the highest sales (~33M)
4.Sales show fluctuation but overall upward trend over time
5.Profit remains stable with slight growth, indicating controlled cost structure
